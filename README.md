## 🎯 Sobre o Projeto

O **YuTube Clone** é uma recriação da interface do YouTube, desenvolvida como projeto prático para aprofundar habilidades em desenvolvimento frontend. O projeto replica fielmente o layout, a navegação lateral, a barra de pesquisa, os cards de vídeo e o design responsivo da plataforma original — utilizando apenas **HTML5**, **CSS3** e **JavaScript puro**, sem frameworks ou bibliotecas externas.

> 💡 O objetivo principal foi dominar conceitos como **Flexbox**, **CSS Grid**, componentização com JavaScript e reprodução fiel de interfaces reais.

## 🚀 Como Executar

Não é necessário instalar nada. Basta clonar o repositório e abrir no navegador.

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/yutube-clone.git

# Entre na pasta do projeto
cd yutube-clone

# Abra o arquivo principal no navegador
# Clique duas vezes em index.html ou use a extensão Live Server no VS Code
```

> 💡 **Dica:** Use a extensão [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) no VS Code para uma experiência com hot reload.

## 📚 O que Aprendi

- Reproduzir layouts complexos com **CSS Grid** e **Flexbox** de forma combinada
- Trabalhar com **variáveis CSS** (`--var`) para manter consistência visual
- Criar componentes reutilizáveis com **JavaScript puro** sem frameworks
- Aplicar **design responsivo** com media queries estratégicas
- Entender a arquitetura visual de grandes plataformas como o YouTube
- Manipulação dinâmica do **DOM** para renderizar listas de conteúdo
